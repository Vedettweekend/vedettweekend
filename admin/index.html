Vedettweekend CMS

Content Management System

Welcome to the Vedettweekend Content Management System

This CMS allows you to easily manage all the content on your website without needing to write code. You can:

  • Edit Featured Artists: Update artist information, images, dates, and social media links
  • Manage Sponsors: Add, remove, or update sponsor logos and information
  • Update Site Settings: Change the logo, background videos, ticket URLs, and more
  • Manage Social Media: Update social media links and icons
  • Create New Pages: Add new content pages to your website
  • Manage Events: Create and update event information

Quick Access

Featured Artists

Manage the artists displayed on your homepage, including images, dates, and social media links.

Sponsors

Update sponsor logos, links, and categories. Control which sponsors appear in the footer.

Site Settings

Change your logo, background videos, ticket URLs, and other global site settings.

Social Media

Update your social media links and icons that appear throughout the website.

Pages

Create new content pages or edit existing ones using the markdown editor.

Events

Manage event information, dates, locations, and ticket links.

Getting Started

To start managing your content:

  1. Click the "Login" button below to access the CMS
  2. Use the left sidebar to navigate between different content types
  3. Click on any item to edit it, or use the "New" button to create new content
  4. Upload images and files using the media library
  5. Save your changes - they will be automatically published to your website

Need Help?

If you need assistance with the CMS:

  • Check the help documentation in the CMS interface
  • Contact your website developer for technical support
  • Most fields have helpful hints and descriptions
  • Images should be uploaded in the correct format (PNG, JPG, SVG)
